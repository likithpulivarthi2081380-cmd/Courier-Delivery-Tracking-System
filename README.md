# 📦 Courier Delivery Tracking System

## 📌 Description
This project is a C++ based application to manage courier records. It allows users to add, track, update, and store courier details using file handling.

## 🚀 Features
- Add courier details
- Track courier by ID
- Update delivery status
- Store data in file

## 🧠 Concepts Used
- OOP (Classes, Inheritance, Polymorphism)
- File Handling
- Pointers

## ▶️ How to Run
1. Compile:
   g++ courier.cpp -o courier
2. Run:
   ./courier
